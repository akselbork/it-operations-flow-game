# it-operations-flow-game
The IT Operations Flow Game is an open systems thinking simulation for IT Operations teams.

The game is designed to help teams experience how limited capacity, rework, delayed information, local optimization, hidden dependencies and poor communication affect reliability across teams.

The material is free to use, adapt and share under the Creative Commons Attribution-ShareAlike 4.0 license.

Paid facilitation, workshops or organization-specific adaptations are allowed, as long as the original project is credited and derivative material follows the same license.
